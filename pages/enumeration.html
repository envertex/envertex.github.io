NMAP

Scan Types

Scan Type Command Notes
UDP Scan (-sU) sudo nmap -sU -sS $IP
sudo nmap -sU -T5 --top-ports 500 $IP 		Use together with TCP SYN scans for full coverage
TCP Connect (-sT) nmap -sT $IP Completes the full TCP handshake
Stealth Scan (-sS) sudo nmap -sS $IP Avoids completing the TCP handshake
Specific Port Scan nmap -p [portNumber] $IP Scan one or multiple specific ports
Network Sweeping (-sn) nmap -sn [IP_range] Host discovery without port scanning
Top 20 Ports nmap --top-ports 20 $IP Most common ports from nmap-services

Detection and Scanning

Detection Type Command Notes
OS Detection (-O) nmap -O $IP Attempts to identify OS type and version
OS Guessing nmap --osscan-guess $IP Guesses OS based on packet behavior
Service Discovery (-sV) nmap -sV $IP Detects running services and versions
Aggressive Scan (-A) nmap -A $IP Includes OS detection, scripts, traceroute (slow)

Nmap Scripts

(Scripts located in /usr/share/nmap/scripts/)

Script Function Command Notes
Run Script nmap --script [scriptName] $IP Executes a specific NSE script
Script Help nmap --script-help [scriptName] Displays script documentation
Example Script nmap --script http-headers $IP Example using the http-headers script
Script Categories nmap --script [category] $IP auth, broadcast, brute, default, discovery, exploit, fuzzer, malware, safe, version, vuln

MSRPC

Nmap Scripting Scan

MSRPC Enumeration

Uses Nmap to enumerate Microsoft RPC interfaces exposed on port 135. This helps identify accessible RPC services that may allow user, group, or domain enumeration. 

nmap -p 135 --script msrpc-enum <target-ip>

Enumerating MSRPC Endpoints with rpcdump

rpcdump lists all registered RPC endpoints and UUIDs on the target. This is useful to detect vulnerable or misconfigured RPC services that can be targeted later. 

rpcdump.py -p 135 <target-ip>

Enumerate RPC over HTTP

Scans port 593 to identify RPC services exposed over HTTP, commonly found in Windows environments and sometimes forgotten by administrators. 

nmap -p 593 --script http-rpc-epmap <target-ip>

rpcclient Enumeration

Null Session Connection

Attempts to connect to the RPC service without credentials. If successful, it allows enumeration of users, groups, and domain information. 

rpcclient -U "" -N <target-ip>

Retrieve Server Information

Displays basic information about the server, such as OS version, domain name, and server role. 

rpcclient -U "" -N <target-ip> -c "srvinfo"

Enumerate Domain Users

Lists all domain users. The output includes usernames and their RIDs, which are required for deeper enumeration. 

rpcclient -U "" -N <target-ip> -c "enumdomusers"

Enumerate Domain Groups

Lists all domain groups and their RIDs. This helps identify privileged or interesting groups for further attacks. 

rpcclient -U "" -N <target-ip> -c "enumdomgroups"

Query Specific User by RID

Retrieves detailed information about a user by specifying its RID obtained from enumdomusers. 

rpcclient -U "" -W "" <target-ip> -c "queryuser <RID>"

Resolve Usernames to SIDs

Translates usernames into Security Identifiers (SIDs), which are useful for mapping users and groups in the domain. 

rpcclient -U "" -N <target-ip> -c "lookupnames <username>"

Retrieve Domain Information

Provides detailed domain information such as domain name, SID, and role (workstation, server, or domain controller). 

rpcclient -U "" -N <target-ip> -c "querydominfo"

SMB / Host Enumeration

Host Enumeration

Nmap basic SMB scan

Escanea los puertos SMB más comunes (139 y 445) para confirmar si el servicio SMB está activo en el host objetivo.

nmap -v -p 139,445 <IP>

SMB OS discovery

Utiliza scripts NSE para identificar el sistema operativo remoto, hostname, dominio y versión de SMB. El flag unsafe permite pruebas más agresivas.

nmap -p 139,445 --script-args=unsafe=1 --script /usr/share/nmap/scripts/smb-os-discovery <IP>

NetBIOS scan

Enumera nombres NetBIOS, equipos Windows y workgroups dentro de una red especificada.

sudo nbtscan -r 192.168.50.0/24

Windows network view

Lista equipos y recursos visibles dentro de un dominio Windows.

net view \\<domainName> /all

Nmap Scripting Scan

Enumerate SMB shares

Enumera los recursos compartidos SMB accesibles en el objetivo.

nmap --script smb-enum-shares.nse -p445 <IP>

Enumerate SMB users

Intenta enumerar usuarios locales o de dominio a través de SMB.

nmap --script smb-enum-users.nse -p445 <IP>

Full SMB enumeration

Enumera dominios, grupos, procesos, servicios, sesiones, shares y usuarios usando múltiples scripts SMB.

nmap --script smb-enum-domains.nse,smb-enum-groups.nse,smb-enum-processes.nse,smb-enum-services.nse,smb-enum-sessions.nse,smb-enum-shares.nse,smb-enum-users.nse -p445 <IP>

SMB vulnerability scan (generic)

Escanea vulnerabilidades SMB conocidas excluyendo ataques DoS, incluyendo verificación de versiones.

nmap -p139,445 --script "smb-vuln-* and not(smb-vuln-regsvc-dos)" --script-args smb-vuln-cve-2017-7494.check-version,unsafe=1 <IP>

SMB vulnerability scan (specific CVEs)

Busca vulnerabilidades SMB críticas como Conficker, MS08-067, MS17-010 (EternalBlue), entre otras.

nmap --script smb-vuln-conficker.nse,smb-vuln-cve2009-3103.nse,smb-vuln-cve-2017-7494.nse,smb-vuln-ms06-025.nse,smb-vuln-ms07-029.nse,smb-vuln-ms08-067.nse,smb-vuln-ms10-054.nse,smb-vuln-ms10-061.nse,smb-vuln-ms17-010.nse,smb-vuln-regsvc-dos.nse,smb-vuln-webexec.nse -p445 <IP>

Check SambaCry version

Verifica si el objetivo es vulnerable a CVE-2017-7494 comprobando la versión de Samba.

nmap --script smb-vuln-cve-2017-7494 --script-args smb-vuln-cve-2017-7494.check-version -p445 <IP>
