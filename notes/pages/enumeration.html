Scan Types

Scan Type Command Notes
UDP Scan (-sU) sudo nmap -sU -sS $IP

TCP Connect (-sT) nmap -sT $IP Completes the full TCP handshake

Specific Port Scan nmap -p [portNumber] $IP Scan one or multiple specific ports
Network Sweeping (-sn) nmap -sn [IP_range] Host discovery without port scanning

Detection Type Command Notes

MSRPC

Nmap Scripting Scan

MSRPC Enumeration

Uses Nmap to enumerate Microsoft RPC interfaces exposed on port 135. This helps identify accessible RPC services that may allow user, group, or domain enumeration. 

nmap -p 135 --script msrpc-enum <target-ip>

Enumerating MSRPC Endpoints with rpcdump

rpcdump lists all registered RPC endpoints and UUIDs on the target. This is useful to detect vulnerable or misconfigured RPC services that can be targeted later. 

rpcdump.py -p 135 <target-ip>

Enumerate RPC over HTTP

Scans port 593 to identify RPC services exposed over HTTP, commonly found in Windows environments and sometimes forgotten by administrators. 

nmap -p 593 --script http-rpc-epmap <target-ip>

rpcclient Enumeration

Null Session Connection

Attempts to connect to the RPC service without credentials. If successful, it allows enumeration of users, groups, and domain information. 

rpcclient -U "" -N <target-ip>

Retrieve Server Information

Displays basic information about the server, such as OS version, domain name, and server role. 

rpcclient -U "" -N <target-ip> -c "srvinfo"

Enumerate Domain Users

Lists all domain users. The output includes usernames and their RIDs, which are required for deeper enumeration. 

rpcclient -U "" -N <target-ip> -c "enumdomusers"

Enumerate Domain Groups

Lists all domain groups and their RIDs. This helps identify privileged or interesting groups for further attacks. 

rpcclient -U "" -N <target-ip> -c "enumdomgroups"

Query Specific User by RID

Retrieves detailed information about a user by specifying its RID obtained from enumdomusers. 

rpcclient -U "" -W "" <target-ip> -c "queryuser <RID>"

Resolve Usernames to SIDs

Translates usernames into Security Identifiers (SIDs), which are useful for mapping users and groups in the domain. 

rpcclient -U "" -N <target-ip> -c "lookupnames <username>"

Retrieve Domain Information

Provides detailed domain information such as domain name, SID, and role (workstation, server, or domain controller). 

rpcclient -U "" -N <target-ip> -c "querydominfo"

SMB

Share Enumeration

smbclient

Lists available SMB shares on the target using anonymous authentication.

smbclient -L //<IP> -U anonymous

Enumerates SMB shares using valid username and password credentials.

smbclient -L //<IP> -U username%password

Connects directly to a specific SMB share using provided user credentials.

smbclient //<IP>/sharename -U username%password

smbmap

Displays accessible SMB shares and permissions for the target host.

smbmap -H <IP>

Enumerates SMB shares using authenticated credentials to reveal permissions.

smbmap -H <IP> -u username -p password

Checks SMB share access using the guest account with no password.

smbmap -H <IP> -u guest -p ''

Recursively lists files and directories inside accessible SMB shares.

smbmap -H <IP> -u username -p password -r

nxc

Enumerates SMB shares using NetExec with valid user credentials.

netexec smb <IP> -u username -p 'password' --shares

Lists available SMB shares by authenticating as the guest user.

netexec smb <IP> --shares -u 'guest' -p ''

User and Group Enumeration

enum4linux

Performs SMB enumeration using supplied credentials to gather user information.

enum4linux -u john -p password123 <IP>

Enumerates users, groups, and shares using anonymous SMB access.

enum4linux <IP>

Runs full SMB enumeration including users, groups, shares, and policies.

enum4linux -a <IP>

Extracts only user account information from the SMB service.

enum4linux -U <IP>

Enumerates only group information available through the SMB service.

enum4linux -G <IP>

Retrieves password policy information from the SMB target.

enum4linux -P <IP>

spider_plus

Recursively downloads accessible SMB files using the spider_plus NetExec module.

netexec smb <IP> -u username -p '' -M spider_plus
