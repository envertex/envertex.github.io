Enumeration

Situational Awareness

This command allows us to view DNS information, IP addresses, gateways, and network configuration.

ipconfig /all

Allows us to identify other devices that are connected to the same network.

arp -a

Displays the routing table of the system, which is very important for lateral movement.

route print

Protection

Allows us to check if the antivirus (Windows Defender) is enabled.

Get-MpComputerStatus

Shows the effective AppLocker rules applied on the system.

Get-AppLockerPolicy -Effective

Checks whether cmd.exe is allowed by AppLocker policies.

Get-AppLockerPolicy -Local | Test-AppLockerPolicy -Path C:\Windows\System32\cmd.exe -User Everyone

System and Processes

Lists running processes and their associated services.

tasklist /svc

Displays environment variables (PATH, shares, profiles, and potentially dangerous paths).

set

Provides detailed information about the operating system and hardware.

systeminfo

Services and Ports

Displays active network connections, listening ports, and associated processes.

netstat -ano

Users

Shows currently logged-in users.

query user

Displays the current username.

echo %USERNAME%

Lists all local users on the system.

net user

Lists all local groups.

net localgroup

Lists members of the local Administrators group.

net localgroup administrators

Lists domain groups (if the system is part of a domain).

net group /domain

Groups and Privileges

Displays the privileges assigned to the current user.

whoami /priv

Lists the groups the current user belongs to.

whoami /groups

Updates

Displays installed Windows updates and hotfixes.

wmic qfe
